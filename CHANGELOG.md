# Changelog

## [1.0.1] - 2026-02-13

### Eliminado

- Eliminado por completo el código, sonidos y comentarios relacionados con el "Himno del Sevilla" en `AudioManager.js`.
- Limpieza de referencias visuales al Sevilla en el `UIManager.js`.

### Cambiado

- Simplificación del sistema de audio: ahora todos los usuarios usan el sonido de notificación estándar del sistema Cyberpunk por defecto.
- Mejora de comentarios internos para mayor legibilidad.

### Agregado

- Restaurado el botón de simulación para el usuario **Ractor09** en el `test-panel.html`.

## [1.0.0] - 2024-01-20

### Agregado

- Lanzamiento inicial de la versión 1.0 del Widget de Chat para OBS.
- Sistema de niveles y rangos.
- Integración con insignias (badges).
- Panel de pruebas (`test-panel.html`).
- Documentación del proyecto.
